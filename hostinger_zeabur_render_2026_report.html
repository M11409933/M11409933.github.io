Hostinger

Website Hosting / Builder
Web hosting WordPress Website Builder VPS Domains Business Email

費用

  • 官方 pricing 頁顯示 Web hosting 起價 $2.99/月（長約期顯示）。
  • Premium 方案示例含：免費網域 1 年、免費 SSL、網站編輯器、每週自動備份等。
  • 也有 Hostinger Horizons / AI 建站、VPS、網域與郵件等產品線。

架站方式

  • 用 拖拉式網站建構器。
  • 直接 WordPress 建站與託管。
  • 使用 Hostinger Horizons：用自然語言描述，AI 幫你做網站/網頁應用。

定位 / 特性

  • 強調「從想法到上線」快速、低技術門檻。
  • 對個人站、品牌站、部落格、小電商很友善。

Zeabur

AI DevOps / PaaS
Free / Dev / Pro / Team AI Hub Templates DNS Email Private Server Mgmt

費用

  • Free：$0；Dev：$5/月（前 14 天免費）；Pro：$19/月（前 14 天免費）；Team：$79/月（3 席）。
  • 官方標示若選擇其雲端伺服器，部分區域（例如 DigitalOcean）伺服器價格 $14 起，且需搭配付費方案。

架站方式

  • 可用 Zeabur Agent 自動偵測語言/框架。
  • 可部署本機專案或模板，也有一鍵模板（WordPress、Supabase、Dify、LobeChat 等）。
  • 支援 DNS、網域、郵件服務與 AI Hub 整合。

定位 / 特性

  • 主打「AI DevOps 工程師」與固定可預期成本。
  • 適合喜歡自動化、模板化、想少管 DevOps 的開發者。

Modern PaaS / Cloud

費用

  • Hobby：$0/月 + compute。

定位 / 特性

  • 強調 production-grade、治理、私網、安全性與可擴展性。
  • 適合 SaaS、AI 產品、工程團隊與需要 SSO / 稽核 / 合規的組織。

比較摘要：你該怎麼選？

情境 建議平台 原因

你是開發者、想用自然語言/模板快速部署，且重視成本可預期 Zeabur AI DevOps 與模板部署很強，對小團隊與自動化流程友善。
