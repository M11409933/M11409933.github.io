上季業績檢討報告（董事會用）

報告日期： 2026-07-14
報告對象： 董事會
資料來源： 內部交易數據分析（Analyst）、總體環境研究（Researcher）

一、摘要

上季業績呈現連續兩季惡化，成交金額較上上季下滑 60.1%（$207,900 → $83,000），成交率同步由 50.0% 降至 30.0%， 平均單筆成交金額亦下滑 40.1%。經分析，下滑主因為內部銷售管道表現嚴重分化—— Cold Call、LinkedIn、Partner 三大管道成交率崩盤至 0% 或低檔，僅 Website 管道逆勢穩定並持續貢獻成交； 同時案件規模同步小型化，形成「成交率、客單價雙重下滑」的複合壓力。案源（Pipeline）本身並未萎縮， 金額規模逐季成長，顯示問題出在轉換與執行端，而非市場需求枯竭。此外，當前總體經濟與政策環境 （景氣調整、關稅與監管趨嚴、採購行為轉趨保守）對一般 B2B 銷售構成不利背景，可作為業績承壓的情境參考， 但目前並無數據可證實其對本公司業績的直接、量化影響，應與內部因素分開看待、不宜混為單一因果解釋。 下季建議聚焦於體檢並修正 Cold Call、LinkedIn、Partner 三管道的銷售流程與名單品質， 同時複製 Website 管道的成功經驗，並密切監控新建商機量放緩此一領先指標。

關鍵觀察：

  • 成交金額連續兩季下滑，且降幅逐季擴大（上上季較前兩季降 45.2%，上季較上上季降 60.1%）。

  • 平均成交金額由 $31,617 降至 $13,833，累計下滑 56.2%，顯示成交案件正快速小型化。
  • 上季為不完整季（僅約當季 84% 時長），實際降幅可能因此略被放大，需下季複核確認。

Pipeline 端： 未結案商機金額規模逐季成長（$64.5K → $92.8K → $105.9K），數量持平微增， 顯示短期內並非「案源枯竭」問題，問題核心在於成交轉換率。惟新建商機量（lead volume）呈下滑趨勢 （28 → 26 → 18，經不完整季調整後約 21.4，仍低於前一季），屬於次要但需留意的領先指標， 若持續下滑，可能影響再下一季的 Pipeline 供給。

三、主因分析

（一）內部因素：管道表現分化為主要驅動力

全期間各管道成交率排名為：Website 70.0% > Event 66.7% > Referral 50.0% > Cold Call 44.4% （客單價最高，達 $32,675）> LinkedIn 33.3% > Partner 22.2%（表現最弱）。

上季各管道出現顯著分化：

  • Cold Call、LinkedIn： 成交率雙雙崩盤至 0%，是拖累整體成交率的最主要因素。
  • Partner： 全程表現最弱且不穩定，上季延續低迷。
  • Event： 客單價急速萎縮，由 $36,467 降至 $6,200，即便成交仍有進帳，但單筆貢獻大幅縮水。
  • Referral： 成交率由 67% 降至 25%，同步走弱。
  • Website： 為唯一逆勢穩定甚至改善的管道（成交率由 67% 提升至 75%），是上季唯一持續貢獻多筆成交的來源。

綜合來看，成交率大幅下滑並非單一原因，而是多管道同步走弱、僅 Website 支撐大局所致； 平均成交金額同步萎縮，形成「量」（成交率）與「質」（單筆金額）雙重壓縮營收的局面。

（二）外部因素：總體環境情境說明（非量化因果，僅供參考）

當前總體經濟與政策情勢呈現多重不利訊號，可作為業績承壓的背景情境：

  • 景氣面： 全球經濟成長放緩、利率維持高檔，企業採購預算轉趨謹慎，資本性與擴張性支出可能延後；製造業景氣疲弱，訂單能見度下降。
  • 競爭面： 跨境比價風氣加劇、定價權承壓，競爭焦點轉向服務與效率，傳統銷售模式效能相對下降，客戶黏著度降低、一次性交易增多。
  • 政策面： 多層次關稅疊加、貿易保護主義升溫，客戶採購決策延後；跨境監管趨嚴推升合規與物流成本、拉長交期。
重要提醒： 上述外部研究反映的是當前總體環境的一般性情勢，與內部交易數據並非取自同一時間基準 （內部數據為近三個季度的相對趨勢，外部研究為近期總體環境觀察），兩者在時間點上不完全對應。 外部環境可合理解釋「大環境不利」的背景，但目前沒有可直接連結兩者的量化證據 （例如特定客戶因關稅延單、特定管道因競爭轉型而流失等具體個案數據）。 因此，外部因素應定位為情境輔助說明，內部管道表現分化與成交率下滑仍是可驗證、 應優先處理的主要且直接的原因。

四、下季建議

  1. 優先體檢 Cold Call、LinkedIn、Partner 三大管道
    針對成交率歸零或長期低檔的管道，檢討名單品質、開發流程、話術與跟進節奏， 判斷問題出在名單源頭（Lead Quality）或執行轉換（Sales Execution），並設定止血或調整的明確時限； 若體檢後仍無改善跡象，應評估縮減資源投入、轉向高效管道。
  2. 複製 Website 管道的成功經驗
    Website 是上季唯一穩定甚至成長的管道，建議拆解其客戶輪廓、內容/流量來源、承接與轉換流程， 評估是否可將部分資源（含行銷預算、業務人力配置）向該管道傾斜，或將其成功要素移植至其他管道。
  3. 關注新建商機量放緩訊號，提前佈局下季 Pipeline
    雖然目前 Pipeline 金額規模仍在成長，但新建商機量已連續下滑， 建議下季初即檢視開發端（Top of Funnel）投入是否需要加碼，避免再下一季出現案源不足的問題。
  4. 檢討案件小型化趨勢，強化高客單價機會的辨識與經營
    平均成交金額大幅萎縮，建議業務端在報價與提案階段強化對高價值機會的辨識與資源投入優先順序， 避免資源過度分散於小型案件。
  5. 因應外部環境，合理調整銷售週期與預期
    在採購決策延後、決策鏈複雜化的大環境下，建議適度拉長銷售週期預估、加強對客戶「價值確定性」的溝通 （如更明確的 ROI 論述、供應鏈金融或效率工具的搭配），並將此情境納入下季業績預測的保守假設中， 但不作為業績下滑的主要歸因。
  6. 下季覆核與資料強化
    上季樣本數偏小（僅 20 筆已結案交易）且為不完整季，降幅可能略被放大， 建議下季重新以完整季資料複核本次結論，並考慮建立管道與客戶層級的外部環境影響追蹤機制， 以便未來能更精確量化外部因素的實際衝擊。

五、待確認事項

以下事項在整合過程中發現資料不足或存在潛在落差，建議相關單位於下次報告前補充：

  1. 【Analyst】樣本數與季度完整性： 上季僅 20 筆已結案交易，且為不完整季（約當季 84% 時長），下滑幅度可能被放大， 建議下季以完整季資料重新驗證本次結論是否成立。
  2. 【Analyst】管道失效的根因未明： 目前僅能確認 Cold Call、LinkedIn、Partner 成交率崩盤的「現象」， 尚無法判斷是名單品質、業務執行、或管道本身特性改變所致， 建議進一步拆解各管道的漏斗數據（如接觸量、初次會議率、報價轉換率）以定位根因。
  3. 【Researcher】外部因素缺乏對本公司的量化連結： 目前的外部環境研究為產業/總體層級的一般性觀察，並未提供可直接對應本公司客戶或管道的具體案例或數據 （例如特定客戶因關稅政策延單的紀錄），建議後續若要在報告中做因果連結，需補充公司層級的佐證資料。
  4. 【時間基準】內部與外部資料的時間對應關係： 本報告已依指示以相對季別（上季、上上季、前兩季）與「當前總體環境」呈現，避免時間軸矛盾， 但仍建議未來建立同一時間窗口下的內外部資料比對機制，以提升情境分析的嚴謹度。
